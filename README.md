# StockView网站

零依赖的股票实时行情网站，仅使用 Python 标准库，无需安装任何第三方包。


浏览器打开 http://47.80.6.12/ ，三个市场独立页面：

| 页面 | 地址 | 默认自选 |
|------|------|----------|
| 美股  | `/us`（首页默认） | 英伟达、特斯拉、谷歌、微软、亚马逊等 |
| 港股 | `/hk` | 腾讯、阿里、美团、小米、中国移动 |
| A股 | `/a` | 贵州茅台、平安银行、中国平安、宁德时代、招商银行 |

环境变量：`PORT`（默认 8000）、`HOST`（默认 0.0.0.0，云平台部署所需）。

## 功能

- 实时价格（每 5 秒自动刷新），美股绿涨红跌，港股和A股红涨绿跌，价格变动闪烁提示
- 每个市场独立自选列表，用户可以自定义自选列表，添加自选个股并更换个股排序，还可以给不同自选个股进行分组（localStorage 按市场分开保存）
- 登录注册功能，用户可以通过注册账户来保存自己关注的自选列表等其他自定义设置
- 添加时校验市场归属：在港股页输入美股代码会提示去对应页面
- 展示涨跌幅、涨跌额、今开、最高、最低、昨收、成交量、成交额、换手率
- 展示美股市场不同板块的实时涨跌幅（包含盘中常规交易时段和盘前，盘后等非常规交易时段）
- 展示各个不同交易时段最强势的板块和最弱势的板块
- 展示链上期权异动大单
- 包含多种不同的语言选择（英语（默认），日语，韩语，简体中文，繁体中文）

## 后续可扩展

- K线图 / 分时图（腾讯有历史数据接口 web.ifzq.gtimg.cn）
- 技术指标（MA、MACD、RSI 等）
- 涨跌提醒、多分组自选
